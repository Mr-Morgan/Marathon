Problem 0.0 "Сумма чисел"
ограничение по времени - 2 секунды
ограничение по памяти - 256 мегабайт
ввод: стандартный ввод
вывод: стандартный вывод

Даны n целых чисел. Выведите их сумму.

Входные данные:
В первой строке содержится единственное целое число n (1 ≤ n ≤ 100) — количество чисел для суммирования. 
Во второй строке содержатся n целых чисел a1, a2, ..., an (1 ≤ ai ≤ 100), разделенных пробелом.

Выходные данные:
В единственной строке выведите одно число — искомую сумму чисел.

Примеры:
Входные данные:
2
5 6
Выходные данные:
11
Входные данные:
4
1 2 3 4
Выходные данные:
10

Problem 1.1 "Телефонные звонки"
ограничение по времени на тест - 1 секунда
ограничение по памяти на тест - 256 мегабайт
ввод - стандартный ввод
вывод - стандартный вывод

В данной задаче вам задана информация о телефонных звонках n человек (в этой задаче гарантируется, что n чётно).
Известно, что всего было совершено n / 2 телефонных звонков, каждый из телефонных звонков характеризуется своим
уникальным идентификатором (целым числом от 1 до 100), в каждом телефонном звонке принимали участие ровно два
разных человека и каждый человек принимал участие ровно в одном звонке.

Перед вами стоит задача определить какие из людей разговаривали друг с другом. Считайте, что люди пронумерованы от 1 до n.

Входные данные

В первой строке следует целое положительное число n (2 ≤ n ≤ 100) — количество людей. Гарантируется, что n чётно.
Во второй строке следует последовательность из n целых положительных чисел a1, a2, ..., an (1 ≤ ai ≤ 100), где ai
равно идентификатору звонка, в котором участвовал человек с номером i. Гарантируется, что каждое из ai-х встречается
в последовательности ровно 2 раза.

Выходные данные

Выведите n строк. В каждой строке выведите по два целых числа — номера людей, которые участвовали в одном и том же
разговоре. Информация о каждом разговоре должна быть выведена ровно один раз. Разговоры, а также номера людей для
каждого разговора разрешается выводить в любом порядке.

Problem 1.2 "Hotels"
ограничение по времени на тест - 1 секунда
ограничение по памяти на тест - 256 мегабайт
ввод - стандартный ввод
вывод - стандартный вывод

Перед началом летнего сезона известное туристическое агенство решило назначить звёздность n отелям, то есть каждому
отелю должно быть присвоено определенное количество звёзд — от 1 до 5. Про каждый отель известен его рейтинг — целое
число от 1 до 100. Все рейтинги различны.

После присвоения всем отелям звёздности для любых двух отелей должно выполняться следующее свойство — если рейтинг
отеля A больше чем рейтинг отеля B, то количество звёзд, присвоенное отелю A, должно быть не меньше, чем количество
звёзд, присвоенное отелю B.

Перед вами стоит задача назначить количество звёзд каждому отелю таким образом, чтобы выполнилось описанное выше
свойство, а также, чтобы отелей каждой звёздности было поровну. Гарантируется, что n кратно пяти.

Входные данные

В первой строке следует целое положительное число n (5 ≤ n ≤ 100) — количество отелей, n кратно 5.

Во второй строке следует последовательность из n различных целых положительных чисел a1, a2, ..., an (1 ≤ ai ≤ 100),
где ai равно рейтингу i-го отеля.

Выходные данные

Выведите n целых чисел от 1 до 5, где i-е число должно быть равно звёздности i-го отеля.
